Основная задача:

Дано поле NxM клеток, конь стоит в левой верхней клетке (0,0). Он может ходить буквой Г (как в шахматах) но только вниз и вправо, то есть всего 4 варианта движения.

Надо вычислить сколько различных способов добраться до клетки (N-1, M-1), если нумерация строк и столбцов начинается с нуля.

Решение динамическим программированием, ленивая рекурсия.

Формат ввода
Входные данные: два целых числа
N - количество строк доски, ограничения 1 <= N <= 50
M - количество столбцов доски, ограничения 1 <= M <= 30
Доска может быть не квадратной, то есть возможны примеры, когда N != M .

Формат вывода
Выходные данные: одно число
S - количество способов добраться конем из клетки (0,0) в клетку (N-1, M-1).
Ограничения: 0 <= S <= 10**16
Обратите внимание на ограничения, число вызовет переполнение типа int, поэтому предлагается использовать long long .
